# Enbridge (ENB) Financial Analysis

A comprehensive financial and technical analysis of Enbridge Inc. (TSX/NYSE: ENB),
completed as part of the DAB401 course.

## What This Project Covers
- Financial ratio analysis (profitability, liquidity, valuation)
- CAPM & Beta calculation using regression
- Free Cash Flow and WACC-based intrinsic valuation
- Stock price technical analysis with Python
- Monte Carlo simulation (100 runs, 30-day forecast)
- FB Prophet price forecasting
- Final investment recommendation (Buy / Hold / Sell)

## Tools & Libraries
Python, Jupyter Notebook, pandas, yfinance, matplotlib,
scikit-learn, Prophet, NumPy, openpyxl
